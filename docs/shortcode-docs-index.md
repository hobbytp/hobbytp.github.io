## 🚀 常见任务流程

### 任务: 创建 Xiaomi 文章并使用 shortcodes

1. ✅ 读 [快速参考](shortcode-quick-reference.md#已实现的-shortcodes) (了解可用功能)
2. ✍️ 写 Markdown 和 shortcodes
3. 🧪 运行 `make dev` 预览
4. ✅ 对照 [快速参考](shortcode-quick-reference.md#🎯-使用场景速查) 验证用法
5. 📤 提交！

---

### 任务: 修改 Alert 样式

1. 📖 读 [快速参考 - Alert 部分](shortcode-quick-reference.md#2️⃣-alert)
2. 🔍 在 `assets/css/custom.css` 找到 `.alert-info` 等样式 (lines 1171-1195)
3. 🎨 修改 CSS
4. 🧪 `make dev` 预览
5. 按照 [测试清单](shortcode-testing-checklist.md#-主题测试) 测试
6. ✅ 完成！

---

### 任务: 创建新 Shortcode

1. 📖 读 [最佳实践 - 创建新 Shortcode 的步骤](shortcode-best-practices.md#创建新-shortcode-的步骤)
2. 🧩 查看 [模板库](shortcode-template-library.md) 找合适的模板
3. 📝 复制模板，修改为你的需求
4. 🗂️ 创建 `layouts/shortcodes/my-shortcode.html`
5. 🎨 在 `assets/css/custom.css` 添加样式
6. 🧪 按照 [测试清单](shortcode-testing-checklist.md) 完整测试
7. 📚 在 [使用指南](shortcode-guide-2025.md) 添加文档
8. ✅ 完成！
